javaScript Troubleshooting: Funtion returning "Undefined"

Why is my console returning "undefined" even though the program is running fine? 😓

First,let us understand why is the console showing an unneccesary "undefined". Functions always return a value. In JavaScript,if no return value is specifies,the function will return "undefined".

Since we now know the "why" of the problem,it is easier to get to the "how?" of the solution. Let's get to it. Before closing the function, just before the closing brackets , just add another line in the function declaration,specifying the return value.

Example::

function(parameterOne , parameterTwo){ //conditions var sum = parameterOne + parameterTwo //conditions return '-----'/ return sum } by this the function returns '-----' and the sum according to what is specified.

NOTE:

If the return value is something you want the console to prompt use the inerted colons-> '' or "" . If the return function is a variable defined before,in the function,do not use the colons.Using colons to declare the variable will just return the name of the variable.
